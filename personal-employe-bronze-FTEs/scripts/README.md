# LinkedIn Watcher - Silver Tier Setup Guide

## 📋 Overview

The LinkedIn Watcher automatically monitors your LinkedIn account for:
- ✅ New messages
- ✅ Connection requests
- ✅ Notifications
- ✅ Sales leads (keyword detection)

When new activity is detected, it creates markdown files in `Needs_Action/` folder.

---

## 🚀 Quick Start

### Step 1: Install Dependencies

```bash
# Navigate to project folder
cd "C:\Users\Dell\OneDrive\Documents\GitHub\silver tier\personal-employe-bronze-FTEs"

# Install Python packages
pip install playwright python-dotenv

# Install Playwright browsers
playwright install chromium
```

### Step 2: Configure Credentials

Edit `.env` file and add your LinkedIn credentials:

```bash
LINKEDIN_EMAIL=your.email@example.com
LINKEDIN_PASSWORD=your_password_here
```

⚠️ **IMPORTANT:** Use your actual LinkedIn email and password!

### Step 3: First Login (Manual)

The first time you run the watcher, you need to login manually:

```bash
# Run in non-headless mode (browser visible)
python scripts/linkedin_watcher.py
```

**What happens:**
1. Browser opens automatically
2. Login to LinkedIn (if session not saved)
3. Complete any CAPTCHA if prompted
4. Session is saved for future runs
5. Watcher starts monitoring

### Step 4: Run in Background (Optional)

For continuous monitoring, run in background:

```bash
# Windows PowerShell
Start-Process python -ArgumentList "scripts/linkedin_watcher.py" -WindowStyle Hidden

# Or use Python directly (terminal stays open)
python scripts/linkedin_watcher.py
```

## ⚙️ Configuration (.env)

| Variable | Default | Description |
|----------|---------|-------------|
| `LINKEDIN_EMAIL` | - | Your LinkedIn email |
| `LINKEDIN_PASSWORD` | - | Your LinkedIn password |
| `LINKEDIN_SESSION_PATH` | `./linkedin_session.json` | Session storage path |
| `LINKEDIN_CHECK_INTERVAL` | `60` | Check every X seconds |
| `LINKEDIN_HEADLESS` | `false` | Run browser in background |
| `LINKEDIN_MONITOR_MESSAGES` | `true` | Monitor messages |
| `LINKEDIN_MONITOR_CONNECTIONS` | `true` | Monitor connection requests |
| `LINKEDIN_MONITOR_NOTIFICATIONS` | `true` | Monitor notifications |
| `LINKEDIN_MONITOR_FEED` | `false` | Monitor feed posts |

## 🎯 Sales Lead Detection

The watcher automatically detects sales leads using these keywords:

```
interested, demo, pricing, buy, purchase, quote, proposal,
contract, deal, opportunity, budget, decision, timeline,
requirements, rfp, rfi, vendor, solution, partnership,
collaboration, meeting, call, discuss, explore, services,
product, enterprise
```

Sales leads are marked as **high priority** and logged in blue.

```markdown

from: John Smith

status: pending
priority: high
created: 2026-03-11T10:30:00

**Received:** 2026-03-11 10:30:00
**Priority:** high
**Status:** pending

---

## Content

Hi, I'm interested in your services. Can we schedule a demo?

---

## Action Required

- [ ] Review and respond
- [ ] Update status when complete

---
*Generated by LinkedIn Watcher - Silver Tier*
```

## 🔧 Troubleshooting

### Browser doesn't open
```bash
# Make sure HEADLESS is false for first run
LINKEDIN_HEADLESS=false
```

### Login fails
```bash
# Check credentials in .env
# Try manual login first to save session
# Clear session file and retry
del linkedin_session.json
```

### CAPTCHA appears
```bash
# Complete CAPTCHA manually in browser
# Session will be saved for future runs
# Wait 5-10 minutes between attempts if blocked
```

### No messages detected
```bash
# Make sure you have messages in LinkedIn
# Check MONITOR_MESSAGES=true in .env
# Look at linkedin_watcher.log for errors
```

3. **Use app password if available** - More secure than main password
