# Qwen Code Configuration

## Skills

This project includes Qwen Code skills for:
- Web browsing with Playwright
- Code review and analysis
- File management

## Preferred Commands

### Python
- Use `python` for running scripts
- Use `pip` for installing packages

### Git
- Commit frequently with descriptive messages
- Use conventional commits format

### Testing
- Run verification scripts before committing
- Check logs in AI_Employee_Vault/Logs/

## Project Structure

```
personal-employee-gold-FTEs/
├── scripts/              # Main automation scripts
├── integrations/         # API integrations
├── AI_Employee_Vault/    # Knowledge management
├── odoo-docker/          # Odoo ERP setup
├── .qwen/                # Qwen skills
└── Needs_Action/         # Task tracking
```
